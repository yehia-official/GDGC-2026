MODULE 16

🌍 Global Attributes

Global attributes can be applied to any HTML element — they're the swiss-army knife of HTML.

📦

data-*

Store custom data on any element. Access via JS with element.dataset.

👁️

hidden

Hides an element (like display:none but semantic). Toggle with JS.

✏️

contenteditable

Makes any element editable inline — like a mini rich-text editor.

🖱️

draggable

Makes an element draggable. Used with the HTML5 Drag & Drop API.

🌐

lang

Declares the language of content. Inherits from <html> unless overridden.

🔄

translate

"yes"/"no" — hints browser translation tools whether to translate this content.

MODULE 17

🖼️ Responsive Images

Modern websites serve different images to different devices — small for phones, large for desktops. This saves bandwidth and improves performance significantly.

📱

srcset

Multiple sources with width descriptors. Browser picks the best one automatically.

📐

sizes

Tells the browser what size the image will display at different layout widths.

🖼️

<picture>

Serve completely different images or formats (WebP/AVIF) based on media queries.

⏳

loading="lazy"

Delays off-screen images from loading until the user scrolls near them.

🔄

decoding="async"

Decodes the image off the main thread so it doesn't block rendering.

📦

WebP / AVIF

Modern formats that are 25–50% smaller than JPEG/PNG with the same quality.

Responsive Image Techniques
<!-- srcset: serve different sizes -->
<img
  src="hero-800.jpg"
  srcset="hero-400.jpg 400w,
          hero-800.jpg 800w,
          hero-1200.jpg 1200w"
  sizes="(max-width:600px) 100vw, 800px"
  alt="Hero image"
  loading="lazy"
  decoding="async"
>

<!-- picture: WebP with JPEG fallback -->
<picture>
  <source srcset="photo.avif" type="image/avif">
  <source srcset="photo.webp" type="image/webp">
  <img src="photo.jpg" alt="Description">
</picture>

<!-- Art direction: different crop per device -->
<picture>
  <source media="(max-width:600px)"
    srcset="portrait-crop.jpg">
  <img src="landscape-crop.jpg" alt="Team">
</picture>

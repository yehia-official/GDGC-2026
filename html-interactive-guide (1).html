👁️ النتيجة الحية:

💡 نصيحة: جرّب تضيف class="desc" على عناصر تانية أو غيّر الـ id!

🎯 اختيار الـ Selectors

/* اختيار بـ ID */ #main-title { color: blue; } /* اختيار بـ Class */ .description { font-size: 1.1rem; } /* اختيار بنوع العنصر */ p { line-height: 1.8; }

🧠 اختبر نفسك

1️⃣ كم ID ممكن يكون في العنصر الواحد؟
2️⃣ كم Class ممكن يكون في العنصر الواحد؟
